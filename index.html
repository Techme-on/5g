Private 5G networks are mobile non-public networks. They can use licensed, unlicensed, or shared spectrum. Private 5G networks are boosting existing networking capabilities. At the same time they introduce new possibilities with use cases that cannot be supported from other systems.

5G in A Box Servicess

Network Slicing Manager

NSM can define slice sizes, different quality of service per slice, traffic rules per slice including prioritization and preemption rules. Through Network Slicing Manager you can manage access right to the network slices in the multi tenant use case. 

Mission Critical Push to Talk

5G Core supports push to talk functionality (MCPTT). Push to talk is a feature that can be used in group communication cases that is required in private networks for high priority personal communications.

IoT gateway

Manage IoT devices with extended battery life and local data processing capability. IoT gateway provides an opportunity to do local break-out for IoT data in the case of centralized core.

Media broadcast server

Mobile networks can also be used in multicast mode. Media broadcast server provides a capability to broadcast video or audio streams over the network area.
